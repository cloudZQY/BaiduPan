## 开发

启动*render*和*main*两个进程

```bash
npm run start-renderer-dev
npm run start-main-dev
```

同时启动两个进程

```bash
npm run start-dev
```

如果你想debug *main*进程
```bash
npm run start-debug
```
然后使用vscode，F5进行debug

## 打包

```bash
npm run dist
```
